Data

Speakers make systematic truth-value judgements: Neil giggles is true or false, not meaningless. The entailment if Neil giggles, someone giggles holds automatically.

New tool
The truth-value test

Given a mini-model \(\mathcal{M} = \langle D,I\rangle\) with domain \(D = \{n,m,f\}\) and denotations assigned to expressions, a sentence \(\phi\) is true iff the truth condition derived from \(\sem{\phi}\) is satisfied in \(\mathcal{M}\).

Historical Frege, Tarski, and the birth of model-theoretic semantics
Background

Frege (1892) distinguished concept (unsaturated function, e.g. giggles) from object (saturated argument, e.g. Neil). Predication is function-application, not a relation between two objects. This makes compositionality possible.

Tarski (1933) defined truth recursively for formal languages: a sentence is true in a model \(\mathcal{M}\) iff a certain set-theoretic condition holds. Montague (1970) and Davidson (1967) extended this to natural language — the framework we still use.

Reflect
1In your own words:

Explain Frege's concept/object distinction. Why does calling giggles a "function" rather than a "property" matter for building a compositional theory?

2Critical engagement:

Tarski's truth-definition was designed for formal languages with no ambiguity or context-dependence. Name one feature of natural language that creates a problem for direct application, and briefly sketch how it might be addressed.

Formal Sets, membership, and predicate logic as our meta-language
Definition

\(\sem{\text{giggles}} = \{\,x : x \text{ giggles}\,\}\) — the extension of a one-place predicate of type \(\langle e,t\rangle\) is the set of individuals satisfying it.

Key idea

Neil giggles is true iff \(\sem{\text{Neil}} \in \sem{\text{giggles}}\). Truth conditions become membership conditions. Set theory is our meta-language.

New tool
Predicate Modification — intersection

When two predicates of type \(\langle e,t\rangle\) are sister nodes, they combine by Predicate Modification: \(\sem{AP\;NP} = \sem{AP} \cap \sem{NP}\). E.g. \(\sem{\text{grey wizard}} = \sem{\text{grey}} \cap \sem{\text{wizard}}\).

Apply it
1 Let \(G = \sem{\text{giggles}} = \{n,f,a\}\) and \(T = \sem{\text{talks}} = \{n,m,a\}\). Compute \(G \cap T\) and \(G \cup T\).
2 Is \(G \subseteq T\)? What would it mean semantically if every giggler also talked?
3 Write \(\sem{\text{giggling talker}}\) using set notation, then list its members.
Typological How does predication vary — and stay the same — across languages?
Overview

Languages encode predication differently: English uses an overt copula (is); Russian drops it in the present; Hebrew and many Creoles use bare juxtaposition. Despite these surface differences, the truth condition — individual ∈ set — is constant.

Cross-linguistic
1Observation:

In Russian, the present-tense copula is typically absent: Нил великан ('Neil [is] a giant'). Does this mean Russian predication has different truth conditions? Justify using the membership model.

2Your own data:

Choose a language you know and describe how a simple predication like Neil is tall is expressed. Does the surface form change the semantic structure? What does your example suggest about the universality of the membership model?
