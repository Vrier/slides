Key idea
Semantic competence is knowing truth conditions — what the world would have to be like for the sentence to be true. A meaning theory must derive exactly these for every sentence.

Frege, Tarski & the model

The concept/object distinction and the recursive truth-definition — the intellectual roots of model-theoretic semantics.
HistoricalLENS 02 / 04

Two founding moves

Frege (1892)
Concept vs. object: the word giggles is an unsaturated concept (a function); Neil is a saturated object (its argument). Predication = applying a function to an argument. This is the move that makes compositionality possible.
Tarski (1933)
Truth for formal languages defined recursively relative to a model \(\mathcal{M} = \langle D, I\rangle\). Meaning becomes truth-in-a-model — the framework we still use today. Applied to natural language by Montague (1970) and Davidson (1967).
Note
Both were logicians working on artificial languages. Extending their frameworks to the messiness of natural language required substantial theoretical work — and remains contested.

Sets as our meta-language

Predicates denote sets; predication is membership; set theory provides ∩, ∪ and complement — all the machinery for composing meanings.
FormalLENS 03 / 04

A predicate denotes a set

Definition
\(\sem{\text{giggles}} = \{\,x : x \text{ giggles}\,\}\) — the extension of a one-place predicate is the set of individuals satisfying it. Type \(\langle e,t\rangle\).
Example
Neil giggles is true iff \(n \in \sem{\text{giggles}}\), taking \(\sem{\text{Neil}} = n\). Truth conditions become membership conditions.
Key idea
Set theory is the right meta-language: nouns and VPs both denote sets, and a subject–predicate sentence asks whether the subject's denotation is a member of the predicate's denotation.
FormalLENS 03 / 04

Worked: Neil giggles

Sentence
\(\text{Neil giggles.}\)
Name denotation
\(\sem{\text{Neil}} = n\)
Predicate denotation
\(\sem{\text{giggles}} = \{\,x : x \text{ giggles}\,\}\)
Truth condition
\(n \in \sem{\text{giggles}}\)
General rule
\(\sem{\alpha\;\beta} = 1 \text{ iff } \sem{\alpha} \in \sem{\beta}\)

Predication across languages

Copulas, bare juxtaposition, verbal marking — different surface forms, one underlying membership model.
TypologicalLENS 04 / 04

Predication strategies

Overt copula
English Neil is tall; Swahili agreement marking. The copula overtly marks predication. Many languages drop it in the present tense (Russian, Arabic, Hebrew).
Zero copula & juxtaposition
Some languages use bare juxtaposition: subject NP + predicate NP with no overt linking element. The predication relation is inferred from linear order and prosody.
Universal
Whatever the surface form, the truth condition is the same: individual ∈ set. The formal membership model abstracts away from morphosyntactic variation.
